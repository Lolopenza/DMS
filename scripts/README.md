# 🛠️ Scripts

Helper scripts for development and maintenance.

## Available Scripts

### Setup & Configuration
- **setup_web.py** - Web setup script
- **check_dependencies.py** - Check if all dependencies are installed
- **check_api_key.py** - Test OpenRouter API key configuration

### Migration (Historical)
- **migrate_core.py** - Migration script (already completed)
- **update_imports.py** - Update imports script (already completed)

### Running
- **run_app.ps1** - PowerShell script to run the app
- **run.ps1** - Alternative PowerShell script

### Legacy
- **gui.py** - Old GUI application (legacy)
- **main.py** - Old main entry point (legacy)

## Usage

```bash
# Check API key
python scripts/check_api_key.py

# Check dependencies
python scripts/check_dependencies.py

# Run app (Windows PowerShell)
.\scripts\run_app.ps1
```
